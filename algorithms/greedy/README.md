# 그리디 (Greedy)

## 핵심 개념

- **매 순간 최적의 선택**
- **교환 논법**: 최적해 증명
- **정렬 활용**: 많은 문제에서 정렬이 핵심

### 그리디 vs DP

- 그리디: 매 단계 최선 → 전체 최선 (증명 필요)
- DP: 모든 경우 고려 후 최선
