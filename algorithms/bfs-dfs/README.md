# BFS / DFS

## 핵심 개념

### DFS (깊이 우선 탐색)

- **구현**: 재귀 or 스택
- **용도**: 경로 탐색, 사이클 검출, 위상 정렬
- **시간**: O(V + E)

### BFS (너비 우선 탐색)

- **구현**: 큐
- **용도**: 최단 거리, 레벨 순회
- **시간**: O(V + E)
