# 그래프 (Graph)

## 핵심 개념

- **그래프 표현**: 인접 리스트, 인접 행렬
- **탐색**: DFS, BFS
- **최단 경로**: 다익스트라, 벨만-포드, 플로이드-워셜
- **MST**: 크루스칼, 프림
- **위상 정렬**: DAG에서 순서 결정

### 시간 복잡도

| 알고리즘 | 시간 복잡도 |
|---------|-----------|
| DFS/BFS | O(V + E) |
| 다익스트라 | O((V + E) log V) |
| 플로이드-워셜 | O(V³) |
| 크루스칼 | O(E log E) |
